## Definition
[java.util.Stack\<E>](https://docs.oracle.com/javase/7/docs/api/java/util/Stack.html)

: bag which supports remove in a last-in, first-out order (LIFO)

**Applications:**

- Parsing
- Expression evaluation
- Depth-first graph searching
- Function call stack
- Recursion

## Runtime

| Method     | Stack  |
| ---------- | ------ |
| push(item) | $O(1)$ |
| pop        | $O(1)$ |
| peek       | $O(1)$ |
