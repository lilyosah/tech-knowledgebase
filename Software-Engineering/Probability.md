Conditional Probability
P(A|B) = P(A, B)/P(B)

Let C -> Aand x -> B

Bayes rule
$class(x) = arg max\dfrac{(P(ci, x))}{P(x))}$
$Ci \in C$

Computing $P(ci, x)$ is very hard, that's where Bayes theorem comes in

Which gets to 
P(C|x) = (P(x|C)P(C))/P(x)
